## The CRAP Score

- [go-crap](https://github.com/padiazg/go-crap) — Go implementation that inspired py-crap (MIT)
- [cargo-crap](https://github.com/Boehs/cargo-crap) — Rust implementation that inspired go-crap
- [gocyclo](https://github.com/fzipp/gocyclo) — Cyclomatic complexity tool for Go (BSD-3-Clause)

## Python Tools

- [mutmut](https://mutmut.readthedocs.io/) — Mutation testing for Python
- [coverage.py](https://coverage.readthedocs.io/) — Code coverage measurement for Python
- [pytest](https://docs.pytest.org/) — Testing framework used by py-crap's coverage runner

## Metrics

- [Cyclomatic Complexity](https://en.wikipedia.org/wiki/Cyclomatic_complexity) — software complexity metric by Thomas J. McCabe Sr.
- [CRAP Score](https://blog.plataformatec.com.br/2014/10/crap-score-what-and-how-it-works.html) — Cyclomatic Readability and Predictability score
